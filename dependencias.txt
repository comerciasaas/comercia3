# INSTALAÇÃO DE DEPENDÊNCIAS - SISTEMA DINÂMICA

## 1. INSTALAR NODE.JS
Baixe e instale o Node.js 18+ de: https://nodejs.org/

## 2. INSTALAR MYSQL
Opção 1 - XAMPP (Recomendado para desenvolvimento):
- Baixe de: https://www.apachefriends.org/
- Instale e inicie Apache + MySQL

Opção 2 - MySQL Standalone:
- Baixe de: https://dev.mysql.com/downloads/mysql/
- Configure usuário root com senha (ou sem senha)

## 3. COMANDOS DE INSTALAÇÃO

# Instalar dependências do projeto
npm install

# Verificar se todas as dependências foram instaladas
npm audit

# Configurar banco de dados
npm run setup-db

# Instalar dependências globais (opcional)
npm install -g nodemon pm2

## 4. VERIFICAÇÃO DE INSTALAÇÃO

# Verificar versões
node --version
npm --version
mysql --version

# Testar conexão MySQL
mysql -u root -p

## 5. INICIAR O SISTEMA

# Terminal 1 - Backend
npm run server:dev

# Terminal 2 - Frontend  
npm run dev

## 6. ACESSAR O SISTEMA
- Frontend: http://localhost:5173
- Backend API: http://localhost:3001
- Teste: http://localhost:5173/teste
